# RegionFlag
Plugin de regiões para servidores Minecraft Paper/Folia (1.21.x). Permite criar áreas no mapa e associar ações automáticas quando jogadores entram ou saem delas — como tocar sons, executar comandos, exibir títulos na tela ou teleportar.  Desenvolvido com foco em performance e compatibilidade total com Folia (multithreading por região).
